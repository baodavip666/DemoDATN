Hướng Dẫn Chọn Size Giày

Để chọn size giày phù hợp cho bạn hoặc con, việc đo chiều dài chân là rất quan trọng. Dưới đây là cách đo chân đúng cách để bạn có thể chọn size giày chính xác:

  • Chuẩn bị một tờ giấy trắng, một cây bút và một cây thước.
  • Đặt chân lên tờ giấy sao cho gót chân và ngón chân dài nhất được đánh dấu rõ ràng.
  • Đo chiều dài từ gót chân đến ngón chân dài nhất (tính theo cm).
  • Sử dụng số đo này để chọn size giày tương ứng với bảng size dưới đây.

Bảng Size Giày Trẻ Em
